Coach Gift Website Handover
===========================

Thank you for choosing Bolu G Services. This package contains all files for your new website, ready for deployment.

Folder Structure:
-----------------
- index.html          -> Homepage and main website content
- assets/css/         -> Stylesheets (if any are moved from inline styles)
- assets/js/          -> JavaScript files (if separated from inline scripts)
- assets/images/      -> All images used on the website
- Handover-Document.pdf -> Official handover certificate
- User-Guide.pdf      -> Instructions on managing and updating your website
- Maintenance-Plan.pdf -> Optional maintenance package recommendations

Website Features:
-----------------
1. Sticky header with brand name and WhatsApp chat button.
2. Hero section with main CTA and headline.
3. Warning section alerting users about business compliance risks.
4. Problem section highlighting urgent business requirements (CAC, TIN, proper documentation).
5. Services section listing all offered services.
6. “Why Choose Coach Gift” section showcasing benefits.
7. Testimonials section with client feedback.
8. FAQ section with expandable answers.
9. Special Offer section with a countdown timer.
10. Final CTA section encouraging immediate action.
11. Footer with company info.

How to Deploy:
--------------
1. Upload all files to your web hosting account (FTP/cPanel/File Manager).
2. Ensure your domain is pointing to the hosting server.
3. Open index.html in a browser to check the website.
4. Refer to User-Guide.pdf for instructions on updating content and images.
5. Change default passwords or credentials immediately for security.
